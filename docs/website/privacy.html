1 Data Collection

KIVO does not collect, store, transmit, or share any personal data — ever. There are no analytics SDKs, no telemetry, no crash reporting, no advertising frameworks, and no tracking of any kind embedded in the application.

We do not know who you are, what files you compress, or how you use the app. This is intentional and by design.

2 File Processing

All compression is performed entirely on your device using local processing only. Your files are never uploaded, transmitted over a network, or shared with any server or third party.

KIVO does not require an internet connection to function. No network requests are made during normal operation.

The original file is never modified or deleted. KIVO reads the source file you select, creates a compressed copy in the location you choose, and does nothing else with it.

3 Permissions

KIVO requests only the permissions strictly necessary to perform compression:

  • Android — READ_MEDIA_IMAGES: Allows you to select image files for compression. Used exclusively to read the file you explicitly choose.
  • Android — READ_MEDIA_VIDEO: Allows you to select video files. Used only for the file you explicitly choose.
  • Android ≤ 12 — READ_EXTERNAL_STORAGE: Provides file selection on older Android versions where granular media permissions are not yet available.
  • Android ≤ 9 — WRITE_EXTERNAL_STORAGE: Allows saving the compressed file on Android 9 and below.
  • iOS / iPadOS: File access is provided through the system file picker. No broad access to your photo library or full storage is requested.
  • macOS / Windows / Linux: File system access is used only to read the file you select and write the output to the location you choose.

No permission is used for any purpose beyond what is described above.

4 Children's Privacy

KIVO does not collect data from anyone, including children under the age of 13 (or the applicable age of digital consent in your jurisdiction). The app contains no features that target children and is safe for all ages.

5 Third-Party Services

KIVO does not integrate with any third-party services, analytics platforms, advertising networks, social media SDKs, or cloud storage providers.

The app relies solely on open-source libraries for compression (FFmpeg, flutter_image_compress, iText/pdf) that operate entirely on-device and make no external network connections.

6 Data Security

Because KIVO never transmits or stores your data outside your device, the risk of a data breach originating from KIVO is zero. Your files remain under your full control at all times and are only accessible by you through your device's normal file management tools.

7 Your Rights

Since KIVO does not collect any personal data, there is no stored data to access, correct, export, or delete. All processed files remain on your device and can be managed using your operating system's standard tools.

8 Policy Changes

If this privacy policy is updated in the future, the new version will be published alongside the corresponding app release and reflected on this page. The "Last updated" date at the top will reflect the revision date.

Continued use of KIVO after a policy update constitutes acceptance of the updated terms.

1 Coleta de Dados

O KIVO não coleta, armazena, transmite ou compartilha nenhum dado pessoal — jamais. Não há SDKs de análise de uso, telemetria, relatório de erros, frameworks de publicidade nem rastreamento de qualquer natureza no aplicativo.

Não sabemos quem você é, quais arquivos você comprime nem como você usa o app. Isso é intencional e parte do design do produto.

2 Processamento de Arquivos

Toda a compressão é realizada inteiramente no seu dispositivo, sem qualquer processamento externo. Seus arquivos nunca são enviados, transmitidos por rede ou compartilhados com servidores ou terceiros.

O KIVO não exige conexão com a internet para funcionar. Nenhuma requisição de rede é feita durante o uso normal do aplicativo.

O arquivo original nunca é modificado ou excluído. O KIVO lê o arquivo de origem que você selecionar, cria uma cópia comprimida no local que você escolher e nada mais.

3 Permissões

O KIVO solicita apenas as permissões estritamente necessárias para realizar a compressão:

  • Android — READ_MEDIA_IMAGES: Permite selecionar arquivos de imagem. Usado exclusivamente para o arquivo que você escolher explicitamente.
  • Android — READ_MEDIA_VIDEO: Permite selecionar arquivos de vídeo. Usado apenas para o arquivo que você escolher.
  • Android ≤ 12 — READ_EXTERNAL_STORAGE: Seleção de arquivos em versões mais antigas do Android onde as permissões granulares ainda não estão disponíveis.
  • Android ≤ 9 — WRITE_EXTERNAL_STORAGE: Permite salvar o arquivo comprimido no Android 9 e anteriores.
  • iOS / iPadOS: O acesso a arquivos é fornecido pelo seletor de arquivos do sistema. Não é solicitado acesso amplo à biblioteca de fotos nem ao armazenamento.
  • macOS / Windows / Linux: O acesso ao sistema de arquivos é usado apenas para ler o arquivo selecionado e gravar a saída no local que você escolher.

Nenhuma permissão é utilizada para finalidade diferente das descritas acima.

4 Privacidade de Crianças

O KIVO não coleta dados de ninguém, incluindo crianças menores de 13 anos (ou a idade de consentimento digital aplicável na sua jurisdição). O aplicativo é seguro para todas as idades e não contém funcionalidades direcionadas a crianças que exijam tratamento especial de dados conforme o ECA, LGPD, COPPA ou GDPR-K.

5 Serviços de Terceiros

O KIVO não se integra a nenhum serviço de terceiros, plataformas de análise, redes de publicidade, SDKs de redes sociais ou provedores de armazenamento em nuvem.

O aplicativo utiliza apenas bibliotecas de código aberto para compressão (FFmpeg, flutter_image_compress, iText/pdf) que operam inteiramente no dispositivo e não fazem conexões externas.

6 Segurança dos Dados

Como o KIVO nunca transmite nem armazena seus dados fora do seu dispositivo, o risco de vazamento de dados originado pelo KIVO é zero. Seus arquivos permanecem sob seu total controle a qualquer momento e são acessíveis apenas por você, através das ferramentas de gerenciamento de arquivos do seu próprio sistema.

Caso você resida no Brasil, seus direitos são garantidos pela Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018). Em razão da ausência total de coleta de dados, nenhuma das obrigações previstas na LGPD se aplica ao KIVO, pois não há operação de tratamento de dados pessoais.

7 Seus Direitos

Como o KIVO não coleta nenhum dado pessoal, não há dados armazenados para acessar, corrigir, exportar ou excluir. Todos os arquivos processados permanecem no seu dispositivo e podem ser gerenciados pelas ferramentas padrão do seu sistema operacional.

8 Alterações nesta Política

Caso esta política de privacidade seja atualizada no futuro, a nova versão será publicada juntamente com o lançamento correspondente do aplicativo e refletida nesta página. A data de "Última atualização" no topo do documento será atualizada.

O uso continuado do KIVO após uma atualização da política implica a aceitação dos novos termos.
